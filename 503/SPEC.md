# 生活效率工具箱 - 项目规范

## 1. 项目概述
- **项目名称**: 生活效率工具箱 (Life Efficiency Toolbox)
- **项目类型**: 单页SPA应用
- **核心功能**: 聚合7个实用工具的生活效率助手
- **技术栈**: 原生HTML5 + CSS3 + JavaScript (ES6+)
- **部署方式**: GitHub Pages直接部署

## 2. 工具选择

### 已选工具清单（共7个）
1. **待办清单** - 任务管理、添加、完成、删除、优先级筛选、数据统计
2. **密码生成器** - 自定义长度、字符类型、批量生成、复制功能、强度检测
3. **简易记账** - 收支记录、分类管理、数据统计、月份报表
4. **单位换算器** - 长度/重量/面积/体积/温度五大类换算、历史记录
5. **随机抽签/抽奖器** - 单个抽取、批量分组、滚动动画、历史记录
6. **番茄专注计时器** - 25分钟专注+5分钟休息、可视化进度环、专注统计
7. **习惯打卡** - 自定义习惯、日历视图、连续天数统计、完成率

## 3. 页面结构

```
├── 顶部导航栏 (8个Tab切换)
│   ├── 待办清单
│   ├── 密码生成器
│   ├── 简易记账
│   ├── 单位换算器
│   ├── 随机抽签/抽奖器
│   ├── 番茄专注计时器
```

## 4. 功能详述

### 4.1 待办清单
- 添加新待办事项（标题+优先级）
- 标记完成/取消完成
- 编辑待办内容
- 删除待办事项
- 筛选（全部/进行中/已完成）
- 统计完成率
- LocalStorage持久化

### 4.2 密码生成器
- 密码长度滑块 (8-64位)
- 字符选项：大写字母、小写字母、数字、特殊符号
- 一键生成
- 一键复制到剪贴板
- 密码强度指示（弱/中/强）
- 历史记录（最近10条）
- LocalStorage持久化

### 4.3 简易记账
- 添加收支记录（金额、类型、分类、备注、日期）
- 收入/支出分类管理（工资、奖金、餐饮、交通等）
- 月度收支统计摘要
- 月度收支可视化图表
- 删除记录
- LocalStorage持久化

### 4.4 单位换算器
- 分类切换：长度/重量/面积/体积/温度
- 输入数值和源单位，自动换算目标单位
- 一键复制换算结果
- 一键清空输入
- 历史换算记录（最近20条）
- LocalStorage持久化

### 4.5 随机抽签/抽奖器
- 自定义候选名单（多行输入，每行一个选项）
- 单个抽取模式：快速随机选中一个
- 批量分组模式：将名单均匀分组
- 抽奖滚动动画效果
- 开奖结果展示
- 抽奖历史记录（最近30条）
- LocalStorage持久化

### 4.6 番茄专注计时器
- 默认25分钟专注 + 5分钟休息
- 自定义专注/休息时长
- 开始/暂停/重置控制
- 圆形进度环动画
- 计时结束Toast提醒
- 今日累计专注时长统计
- 累计启动次数统计
- LocalStorage持久化

### 4.7 习惯打卡
- 自定义添加习惯（早起、读书、运动等8种图标）
- 日历视图展示每日打卡状态
- 点击切换打卡状态，已打卡高亮
- 自动计算连续打卡天数
- 月度打卡完成率统计
- 习惯删除管理
- LocalStorage持久化

### 4.8 关于我
- 个人信息展示（姓名、职位、地区）
- 个人简介
- 技能特长标签
- 项目经验列表
- 联系方式

## 5. 全局功能

### 5.1 主题切换
- 深色/浅色模式一键切换
- 记住用户偏好（LocalStorage）
- 右上角月亮/太阳图标

### 5.2 数据管理
- 全部数据JSON导出
- JSON文件导入恢复数据
- 数据格式校验
- 导入导出包含全部7个工具的数据

### 5.3 使用统计面板
- 总使用次数
- 各工具使用频次（7项）
- 待办完成率
- 待办总数
- 本月收支/支出/余额
- 番茄钟今日专注时长
- 习惯打卡统计

### 5.4 快捷入口
- 常用功能快捷下拉菜单
- 最近使用记录（最近3条）
- 点击快速跳转到对应工具

### 5.5 键盘快捷键
- Ctrl/Cmd + 1-8：切换Tab
- Ctrl/Cmd + D：切换主题
- Ctrl/Cmd + E：导出数据
- Escape：关闭下拉菜单

## 6. 视觉动效
- 加载动画（工具切换时）
- 点击操作反馈（按钮波纹效果）
- 页面过渡动效（淡入淡出）
- 数字滚动动画
- 抽奖滚动动画
- 番茄钟进度环动画

## 7. 响应式设计
- 桌面端（>1024px）：标准布局
- 平板端（768px-1024px）：自适应收缩
- 移动端（<768px）：单列布局，触控优化
- 导航Tab小屏只显示图标
