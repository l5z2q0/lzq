# 生活效率工具箱

一款纯原生 HTML + CSS + JavaScript 构建的生活效率工具箱，支持7大实用功能，开箱即用，数据本地存储，部署简单。

## 功能清单

### 待办清单
- 添加/编辑/删除待办事项
- 支持普通/重要/紧急三种优先级
- 筛选显示：全部/进行中/已完成
- 实时统计待办数量和完成率
- LocalStorage 持久化存储

### 密码生成器
- 自定义密码长度（8-64位）
- 可选字符类型：大写字母、小写字母、数字、特殊符号
- 实时密码强度指示
- 一键复制到剪贴板
- 最近10条历史记录

### 简易记账
- 收入/支出分类管理
- 月度账目筛选
- 收支统计摘要
- 可视化图表展示
- 备注信息记录
- LocalStorage 持久化存储

### 单位换算器
- 支持长度/重量/面积/体积/温度五大类单位换算
- 实时换算结果显示
- 一键复制换算结果
- 历史记录保存
- 操作次数统计

### 随机抽签/抽奖器
- 单个抽取模式：快速随机选中一个
- 批量分组模式：将名单均匀分组
- 滚动抽奖动画效果
- 开奖结果展示
- 抽奖历史记录

### 番茄专注计时器
- 默认25分钟专注 + 5分钟休息
- 自定义专注/休息时长
- 圆形进度环动画
- 计时结束弹窗提醒
- 今日累计专注时长统计

### 习惯打卡
- 自定义添加习惯（早起、读书、运动等）
- 日历视图打卡展示
- 连续打卡天数统计
- 月度打卡完成率
- 习惯删除管理

### 全局功能
- **深色/浅色主题切换**：一键切换，自动记忆偏好
- **数据导入/导出**：JSON格式备份，一键恢复
- **使用统计面板**：展示7大工具使用频次数据
- **快捷入口**：快速访问常用功能和最近记录
- **键盘快捷键**：
  - `Ctrl/Cmd + 1-8`：切换 Tab
  - `Ctrl/Cmd + D`：切换主题
  - `Ctrl/Cmd + E`：导出数据
  - `Escape`：关闭菜单

### 响应式设计
- 完美适配桌面端、平板端、移动端
- 触控操作友好
- 移动端自适应布局

## 部署说明

### GitHub Pages 部署

1. **创建仓库**：在 GitHub 创建新仓库 `life-toolbox`

2. **上传代码**：将整个项目文件上传到仓库

3. **开启 Pages**：
   - 进入仓库 `Settings`
   - 左侧菜单选择 `Pages`
   - Source 选择 `Deploy from a branch`
   - Branch 选择 `main` `/ (root)`
   - 点击 `Save`

4. **访问网站**：等待 1-2 分钟，访问 `https://你的用户名.github.io/life-toolbox/`

### 本地运行

直接在浏览器中打开 `index.html` 文件即可使用，无需任何服务器。

## 技术特性

- **纯原生实现**：无任何框架依赖，轻量高效
- **模块化编码**：各功能模块分离，易于维护
- **ES6+ 语法**：使用 class、箭头函数、模板字符串等现代语法
- **CSS 变量**：支持主题切换，方便定制
- **LocalStorage**：数据本地持久化，保护隐私
- **语义化 HTML**：良好的可访问性支持
- **响应式布局**：移动端适配优化
