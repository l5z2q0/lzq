# 生活效率工具箱 🧰

一个集成多种实用工具的网页应用，帮助用户提升日常生活效率。

## ✨ 功能清单

### 📝 待办清单
- 添加、删除待办事项
- 标记完成状态
- 筛选全部/待完成/已完成
- 清空已完成任务
- 导出待办数据

### 🧮 计算器
- 基础四则运算
- 百分比计算
- 正负号切换
- 实时显示计算结果

### 💰 记账本
- 记录收入和支出
- 自动计算结余
- 查看收支统计
- 导出记账数据

### 📊 使用统计
- 待办完成率统计
- 计算器使用次数
- 收支汇总图表
- 数据导入/导出

### 👤 关于我
- 个人信息展示
- 实训经历介绍
- 项目技术栈说明

## 🎨 特色功能

- **深色/浅色主题切换** - 一键切换主题，保护眼睛
- **数据持久化** - 所有数据存储在本地，刷新不丢失
- **响应式设计** - 完美适配手机、平板、电脑
- **动画效果** - 流畅的页面切换和交互反馈

## 🛠️ 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计
- **JavaScript** - 交互逻辑
- **Tailwind CSS** - 快速样式开发
- **Font Awesome** - 图标库
- **localStorage** - 本地数据存储

## 🚀 快速开始

### 本地运行

直接用浏览器打开 `index.html` 文件即可运行：

```bash
# 克隆仓库
git clone <仓库地址>

# 进入项目目录
cd <项目目录>

# 使用浏览器打开
start index.html
```

### GitHub Pages 部署

1. 将代码推送到 GitHub 仓库
2. 进入仓库 Settings -> Pages
3. 选择 `main` 分支，点击 Save
4. 等待几分钟后访问：`https://<用户名>.github.io/<仓库名>/`
