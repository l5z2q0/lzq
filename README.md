# 🛠️ 实用工具箱

一款纯本地运行的多功能在线工具集合，所有数据存储在浏览器本地，**100% 隐私安全**，数据不会上传到任何服务器。

## ✨ 核心特点

| 特点 | 说明 |
|------|------|
| 🔒 **隐私安全** | 所有数据存储在浏览器 localStorage，不上传服务器 |
| 🎨 **深色/浅色模式** | 支持一键切换主题，自动记忆用户偏好 |
| 💾 **数据持久化** | 自动保存所有工具数据，刷新页面不丢失 |
| 📤 **数据导入导出** | 支持备份和恢复所有工具数据 |
| 📱 **响应式设计** | 完美适配手机、平板、桌面端 |
| ⚡ **无需依赖** | 纯原生 JavaScript，无任何外部框架依赖 |

## 🚀 快速开始

### 在线预览
访问 [实用工具箱](https://l5z2q0.github.io/lzq/) 即可使用

### 本地运行

**方式一：使用 Node.js**
```bash
# 克隆仓库
git clone https://github.com/l5z2q0/lzq.git

# 进入项目目录
cd lzq

# 使用 http-server 启动（需要先安装 http-server）
npx http-server -p 3000

# 或使用 serve 启动
npx serve . -l 3000

# 访问 http://localhost:3000
```

**方式二：使用 Python**
```bash
# 进入项目目录
cd lzq

# 启动 Python HTTP 服务器
python -m http.server 8000

# 访问 http://localhost:8000
```

**方式三：直接打开**
```bash
# 在文件浏览器中双击 index.html 文件
# 或右键 → 用浏览器打开
```

## 💾 数据备份与恢复

### 导出数据
1. 点击右下角「📥」按钮
2. 浏览器会自动下载 JSON 备份文件
3. 文件命名格式：`工具箱备份_YYYYMMDD.json`

### 导入数据
1. 点击右下角「📤」按钮
2. 选择之前导出的 JSON 备份文件
3. 系统会自动恢复所有工具数据
4. 导入成功后会显示确认提示

### 备份包含的数据
- � theme: 主题设置（深色/浅色模式）
- 📊 usage: 工具使用次数统计
- ✅ todos: 所有待办事项
- 📝 inputs: 所有工具的输入内容
